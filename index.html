WordPress website

Here a developer need not have to code from scratch to create a required clients website. Instead a developer make use of already existing template and plugins to meet the requirement.



Static Website

A static website contains Web pages with fixed content. Static sites are the most basic type of website and are widely used to portray clients services. Websites which shows the fixed content like information about a company, product description, goal etc... all come under static website.

Virtual Website

A website where interaction of user is required to process the need comes under virtual website. For an instance when you have to purchase any product from flipkart(e-commerce website) you have to select that product first after that you need to choose location for the delievery, Name, payment mode etc... must be submited to get that product. In all this process the interaction of user is happening where all the data given by user gets saved in Database for process to happen.
